# Genc Systems Project Bundle

This ZIP contains three separate deployable sites:

- `gencsystems-site/` → main Genc Systems website
- `linden-bloom-florist/` → florist storefront demo
- `hearthstone-plumbing/` → home services demo

## Suggested Vercel setup

1. Deploy `gencsystems-site/` to the main domain: `gencsystems.com`
2. Deploy `linden-bloom-florist/` to: `lindenbloom.gencsystems.com`
3. Deploy `hearthstone-plumbing/` to: `hearthstoneplumbing.gencsystems.com`

## Main portfolio links

The portfolio page is already updated so the florist card opens the florist subdomain and the plumbing card opens the home-services subdomain.

## Note

Each folder is intended to be imported into Vercel as a separate project.
